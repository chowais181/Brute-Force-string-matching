In order to test the algorithm 
1) Go to algoassignment3 -> algoassignment3 -> bin -> Debug -> algoassignment3.exe
2) Choose the algorithm you want to use/test.
3) Search the words based on the given options.
4) After the execution of all the files, you will be presented with the time taken to search through all 
the words in all the files.
5) In "Brute Force" window, the program will display time taken of
   i) Each file individually 
  ii) All the files collectively (keep pressing Enter key untill all 10 files have been processed)
6) The program will automatically use the text files provided by the teacher.
